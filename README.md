# Nunes Sports

Um sistema para exibição, edição e deleção dos produtos vendidos pela companhia.

## Frameworks e Bibliotecas

> Front-end

- ReactJs
- Axios

> Back-end

- MySQL
- Express
- Nodemon
- Cors

## Inicialização do Projeto

> Front-end

```
npm install
npm run dev
```

> Back-end

```
cd server
npm install
npm run dev
```

## Funcionalidades

- Criar produtos
- Editar produtos
- Atualizar produtos
- Deletar produtos
